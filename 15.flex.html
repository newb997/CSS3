Flex

기본 축을 기준으로 정렬함

        * 부모요소 : flex-container
            - display : flex 
            - flex-direction : row(기본값) | column | row-reverse | column-reverse  
            - justify-content : 메인축 정렬(flex-direction이 row라면 row, column이라면 column)
            - align-items : 크로스축 정렬(flex-direction이 row라면 column, column이면 row)
            - flex-wrap : 줄바꿈 여부
            - align-content : 여러줄인 경우에 크로스 축 정렬

        * 자식요소(정렬의 주체) : flex-item
            - flex-grow : 빈 여백을 아이템들의 주어진 비율대로 늘어나면서 레이아웃 영역을 채움
            - flex-shrink : 넘친 아이템 영역을 분해해서 레이아웃 영역안에 배치되도록 하는 방식
            - flex-basis : 아이템의 크기를 직접 설정
